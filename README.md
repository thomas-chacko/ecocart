# EcoCart

Sustainable e-commerce platform with Next.js 16, MongoDB, and Cloudinary.

## Features

- Landing page with dark mode
- Admin authentication (JWT)
- Product CRUD operations
- Image upload with Cloudinary
- Shopping cart with localStorage

## Tech Stack

Next.js 16, MongoDB, Mongoose, JWT, Cloudinary, Tailwind CSS, Framer Motion

## Setup

1. `npm install`
2. Copy `.env.example` to `.env`
3. `node scripts/create-admin.js`
4. `npm run dev`

## Scripts

- `npm run dev` - Development server (port 5555)
- `npm run build` - Production build
- `npm start` - Production server
